Minimum with the current value using an unsigned comparison. T must be an unsigned integer type.

The stabilized version of this intrinsic is available on the [atomic] unsigned integer types via the fetch_min method by passing [Ordering::SeqCst] as the order. For example, [AtomicU32::fetch_min].
