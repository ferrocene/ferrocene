Tool Options
============

``rustfmt`` is qualified exclusively for the following command line options:

* Users shall pass command line option ``--emit files`` to each invocation of ``rustfmt``.
* Users shall pass command line option ``--edition 2021`` to each invocation of ``rustfmt``.
